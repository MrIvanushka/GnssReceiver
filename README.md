# GNSS Receiver

`GNSS Receiver` - это набор библиотек, предназначенных для построения модели навигационного приёмника. На данный момент, в репозитории присутствует только парсер навигационных сообщений, но со временем он обрастёт и другими модулями.

Сейчас логгер, контекст и статистика находятся в проекте `FrameParser`, но совсем скоро они окажутся в отдельном проекте, который будут подключать к себе остальные модули.

## Список библиотек

### FrameParser

`FrameParser` принимает на вход набор отчётов (мнимую часть), полученных в результате трекинга сигналов навигационных спутников, производит синхронизацию бит, подкадров и выделяет полезную информацию. На данный момент `FrameParser` способен обрабатывать только данные GPS и читать только оперативную информацию (все типы сообщений, кроме альманаха).

Совсем скоро будет добавлена статистика по принятым подкадрам и ошибкам синхронизации.

## Примеры

Варианты использования библиотек описаны в примерах. Для их сборки, напишите в консоли:

```
  $ cd _examples/<example>
  $ mkdir _build
  $ cd _build
  $ cmake ../
```

Некоторые примеры могут опираться на txt-файлы с данными. Данные файлы лежат в папке с исходниками примера, их следует переместить в папку со сборкой или в папку с бинарным файлом (в случае запуска не через IDE).
